B. The Moon golf
time limit per test 1 second
memory limit per test 256 megabytes

Условие
Игра в гольф. Вместо лунок - кратеры, вместо мячей - метеориты.
Метеориты имеют разную массу. 
В соответствии с правилами лунного гольфа:
1. Игрок всегда расположен в точке (0, 0);
2. Кратеры имеют форму идеальных кругов;
3. Когда метеорит не забивается в центр кратера, а падает хотя бы у его границы - метеорит неизбежно скатывается в центр и считается забитым;
4. Кратеры не перекрывают и не касаются друг друга;
5. Нельзя забивать в один кратер более одного метеорита;
6. Точка (0,0), в которой находится игрок, не находится внутри кратера.

Максимальное расстояние, на которое игрок может закинуть метеорит зависит от массы m этого метеорита и рассчитывается по формуле:
d = sqrt((10^6) / m).

Вопрос
Задача игрока - забить метеориты в кратеры так, чтобы суммарная масса забитых метеоритов была наибольшей. Для простоты предположим, что игрок не умеет промахиваться, и стратегия просто описывает, какой метеорит в какой кратер нужно забить.

На входе
Первая строка: количество метеоритов n (от 1 до 10^4).
Вторая строка: n чисел (от 1 до 10^6) - массы метеоритов.
Третья строка: количество кратеров k (от 1 до 10^5).
Затем идут k строк, каждая из которых состоит из трех чисел x y r: x и y (от -2000 до 2000) - координаты кратера, r (от 1 до 10) - радиус кратера.

На выходе:
Первая строка должна содержать целое число r (0≤r≤min(n,k)) - количество метеоритов, которые должны быть зачтены. Затем r строк, каждая из которых содержит два целых числа s (1≤s≤n) и t (1≤t≤k) - номер метеорита и номер кратера, в котором он должен быть забит. Метеориты и кратеры нумеруются в порядке появления в исходных данных. Нумерация начинается с единицы.

Examples
input
3
1 100 10000
3
0 10 1
0 100 1
0 1000 1
output
3
1 3
2 2
3 1

input
2
2 3
2
1000 0 1
0 1000 1
output
0
