A. Number System
time limit per test 1 second
memory limit per test 256 megabytes

Условие
Даны два числа и сумма этих чисел, записаные в неизвестной системе счисления. 

Вопрос
Можно ли однозначно определить по этим двум числам и их сумме, в какой системе счисления они записаны?

На входе
Два числа и их сумма. Числа могут состоять из цифр от «0» до «9» и из букв от «A» до «Z».
Числа и сумма могут иметь разную длину, но не больше чем 256 цифр.
На выходе
Если однозначное восстановление системы счисления невозможно, выведите 0, в противном случае - выведите целое положительное основание системы счисления.

Examples
input
A
6
10
output
16

input
1
2
3
output
0

input
Y
Z
output
0
