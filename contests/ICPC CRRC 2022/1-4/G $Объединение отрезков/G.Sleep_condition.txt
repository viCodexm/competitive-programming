G. Sleep
time limit per test 1 second
memory limit per test 256 megabytes

Условие
Есть n записей. Каждая запись выглядит так: hh1:mm1:ss1 - начало сна, и hh2:mm2:ss2 - конец сна.
Есть Вероника, которая спит в это время.

Вопрос
Рассчитать, сколько времени (в секундах) Вероника не спит.

На входе
В первой строке задано одно целое число n (от 1 до 10^5) - количество интервалов сна.
Далее в каждой строке приводится время сна Вероники, разделенное пробелами hh1:mm1:ss1, hh2:mm2:ss2 (hh1, hh2 от 0 до 23, mm1, mm2, ss1, ss2 от 0 до 59) - время засыпания и время пробуждения.
Интервал сна каждой Вероники гарантированно меньше 24 часов.
На выходе:
В одной строке выведите сколько времени Вероника бодрствует в секундах.

Examples
input
5
13:00:00 14:00:00
16:00:00 17:00:00
19:00:00 20:00:00
01:00:00 05:00:00
04:00:00 05:00:00
output
61200

input
4
22:49:13 05:22:28
18:50:30 14:32:25
11:34:33 11:28:25
08:55:57 19:49:53
output
0

input
2
10:48:15 07:42:04
03:00:33 04:52:08
output
11171
